HDX Tools, including Quick Charts, HXL Tag Assist and Data Check, aims to help humanitarians to clean and visualize data. The HDX Tools domain is not a secure service. Users should not use HDX Tools to process personal or otherwise sensitive data. Data uploaded to HDX Tools is not retained within the HDX infrastructure, while data downloaded by HDX Tools from public URLs is cached only as long as necessary for processing.
